# HPLC-Based Analysis of Enzyme Activity and Thermostability
Python-based analysis of 700+ HPLC measurements investigating the effects of co-solvents and surfactants on enzyme performance, thermostability, and biocatalytic process optimization.

## Project Overview

This project investigated how reaction additives influence the performance of an ω-transaminase biocatalyst.

Using Python-based data analysis, more than 700 HPLC measurements were processed and analyzed to identify optimal reaction conditions, quantify enzyme stability, and support process development decisions.

The work was conducted at the Research Center for Enzymology and Applied Biocatalysis and was presented at multiple scientific conferences.

## Research Questions

- Which co-solvents maximize enzyme activity?
- How do surfactants affect thermostability?
- Which reaction conditions achieve the highest conversion?
- What trade-offs exist between activity and stability?
- How can experimental results support process scale-up?
